产品需求文档 (PRD)：随机睡前放送 (Sleepy Randomizer)

版本： V2.0 (Stable)

日期： 2026-03-02

1. 产品定位与目标

一个随机视频和音乐播放 Web 工具。

睡前不建议玩手机的一小时可以看些放松的视频、听些轻松的音乐，当然纸质书也是一个选择……不过对于总是不甘心去睡觉的人来说（比如说我），从电子设备上的内容切换开始，可以更平滑地过渡到睡前的轻松缓慢模式。支持从自定义列表中随机抽取，增加正反馈。如果不想随机，也可以在列表中选择视频播放。
2. 页面架构与视觉逻辑

单页面架构 (SPA)： 采用单页面 Tab 切换（视频池 / 音乐盒），确保紫色渐变背景不中断，维持沉浸感。

设计语言：Glassmorphism (玻璃拟态)。

背景：深紫色浸润渐变背景。

字体规范：页首可采用具有“俏皮活力感”的非衬线字体（如 PingFang SC 或 Inter），增强个性化表达。

布局稳定性 (Layout Stability)：

固定纵横比：包含标题和播放器的中央容器锁定为 16:9。

防抖动：无论抽取的内容长度或视频比例如何，外层容器尺寸严格固定，禁止产生视觉跳动。

响应式策略：

桌面端：播放器与列表分列/并排显示。

移动端：列表强制转换为 Bottom-sheet (底部抽屉) 交互，支持点击展开。
3. 核心功能模块

3.1 视频视图 (Video View)

播放器： 居中展示，当前支持 Bilibili 的视频。点击“随机”后新开页面跳转到该视频播放页面。

随机按钮： 视觉重心，触发 JS 随机算法从 video_list 抽取内容。

提示框建议用户搭配网易云音乐电脑端应用等实现定时关机。

列表抽屉： 卡片式布局，展示视频封面、标题、来源标签及自定义备注。

3.2 音乐视图 (Music View)

独立性： 与视频库物理隔离，拥有独立的随机逻辑与列表。

列表样式： 参考 Spotify 网页端歌单。

备注系统： 允许为每一首歌单独撰写推荐语。

3.3 列表

列表支持展示封面、标题、来源及自定义备注。

折叠控件：参考 B 站网页版，使用 ^ (展开) 与 v (收起) 图标，并配置平滑的过渡动画（图标顺/逆时针旋转180度完成切换）。

添加视频/音乐：点击“添加”按钮，通过粘贴已复制的链接，把对应的视频/音乐增加到列表里。

管理隐藏逻辑：

默认状态下隐藏“删除”、“批量删除”按钮。

需点击列表右上角的“操作/管理”图标方可滑出操作菜单，保持 UI 极简。

分享列表： 位于列表左下方，点击后自动复制“标题+链接+备注”并生成分享卡片。

OneTab 式导出：支持一键生成并导出极简 HTML 备份文件（包含“链接 + 名称”）。
4. 技术与设置 (Settings)

URL 解析增强：自动过滤 URL 中的冗余追踪参数（如 spm_id_from, track_id），仅提取核心 ID 存入 LocalStorage。

智能互斥播放： 切换至视频Tab并未播放时，音乐不自动暂停，视频开始播放音乐才暂停；切换至音乐时，视频保持暂停。

智能去重算法：在当前会话周期内，系统将记录已播放视频，确保在列表循环结束前不出现重复内容。

无障碍设计：针对视障和听障的潜在用户优化，包括但不限于确保色彩使用符合WCAG规范、为图标和图片提供替代文本。
